# 🛠️ 插件与技能系统 (Plugin & Skill System)

欢迎来到“方舟之心”的创意工坊！本目录是宠物所有“**主动技能**”（即工具调用能力）的家园。通过创建插件，你可以教会你的宠物新的魔法，让它为你打开应用、查询信息，甚至与外部世界进行全新的互动。

## ✨ 设计哲学 (Design Philosophy)

- **🔌 可插拔 (Pluggable):** 每个技能都是一个独立的、可被动态加载的模块，添加或移除一个技能，不应影响核心程序的稳定。
- **🔐 安全第一 (Safety-First):** 所有插件都在一个受限的“**沙箱**”内执行。它们能做什么，完全由其在`plugin.json`中明确声明的“**权限清单**”来决定。未经用户明确授权，插件无法访问敏感信息或执行危险操作。
- **📜 契约驱动 (Contract-Driven):** 每个插件都必须遵循一套简单的规则和接口，以确保它们能被“大脑”正确地理解和调用。
